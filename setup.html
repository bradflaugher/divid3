Set as default

divid3 as your search engine

The search router lives at this URL:

https://divid3.com/?q=%s

Most modern browsers auto-detect divid3 the first time you visit divid3.com — they pick up an OpenSearch description and offer the engine under Settings → Search engines. If yours didn't, the steps below take a few seconds.

Choose your browser

Chrome / Edge / Brave / Opera recommended
  1. Open chrome://settings/searchEngines (Edge: edge://settings/searchEngines).
  2. Under Site search, click Add.
  3. Name: divid3. Shortcut: d. URL: paste the URL above.
  4. Click the ⋮ next to the new entry → Make default.

Tip: with the shortcut set to d, you can type d + Tab in the address bar to search divid3 without making it the default.

Arc
  1. Open Arc → Settings → Profiles (or press ⌘+,).
  2. Scroll to Search engine → Manage search engines.
  3. Click Add, fill in: name divid3, shortcut d, URL from above.
  4. Three-dot menu → Make default.
Firefox

Firefox should auto-detect divid3 when you visit divid3.com — look for the + badge on the address bar's magnifying-glass icon, or right-click the address bar.

To set as default afterwards:

  1. Open Settings → Search.
  2. Under Default Search Engine, choose divid3.

Manual add (if auto-detect didn't fire):

  1. Open Settings → Search → Search Shortcuts → Add.
  2. Use the URL above. Firefox replaces %s automatically.
Safari (iOS / iPadOS / macOS)

Safari doesn't expose a way to add an arbitrary search engine natively. Two reliable options:

Option 1 — Customize Search Engine extension

  1. Install Customize Search Engine from the App Store (free, no account).
  2. iOS: Settings → Apps → Safari → Extensions → enable it.
    macOS: Safari → Settings → Extensions → enable it.
  3. Open the extension and paste the URL above.
  4. Set it as the default search.

Option 2 — Add to Home Screen (iOS)

  1. Open divid3.com in Safari.
  2. Tap the Share icon → Add to Home Screen.
  3. Launch from the home-screen icon to get a full-screen, app-like experience that bypasses Safari's chrome.

For best stability on iOS, give the page a few seconds on first visit so the model can finish caching to disk. After that, divid3 works offline.

DuckDuckGo Browser
  1. Open Settings → Search.
  2. Default Search Engine → Other → paste the URL above.
Android (Chrome / Samsung Internet)

Chrome: Visit divid3.com, then open Settings → Search engine → Recently visited and pick divid3.

Samsung Internet: Settings → Search engine → Add search engine and paste the URL above.

Power-user shortcuts

Once divid3 is your default, you can still skip its model entirely with bang shortcuts — they short-circuit straight to the engine:

  • !yt lofi → YouTube
  • !gh react hooks → GitHub
  • !w integral of x^2 → Wolfram Alpha
  • !a usb-c cable → Amazon
  • !m coffee near me → Google Maps
  • !p quantum gravity → Perplexity
  • !g write a haiku → Grok
  • !i black hole → Bing Images
  • !d climate news → DuckDuckGo

Plus typing a bare domain (github.com, localhost:3000) goes straight there. The full list lives in the README.
